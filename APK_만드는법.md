# APK 파일 변환 가이드 (3분 완성)

제 컴퓨터에서는 APK를 직접 생성할 수 없지만, 모든 코드는 준비되었습니다!
아래 방법으로 무료로 즉시 변환하세요.

## 1. 앱 배포 (가장 쉬운 방법)

이 폴더(`car`)를 **Netlify**나 **GitHub Pages** 같은 무료 호스팅에 올립니다.
(이미 올리셨거나 웹 주소가 있다면 2번으로 넘어가세요)

## 2. PWA Builder 접속

1. [PWABuilder.com](https://www.pwabuilder.com/) 사이트에 접속합니다.
2. 내 웹 앱의 **URL(주소)**을 입력하고 `Start`를 누릅니다.
3. 잠시 분석이 끝나면 **"Package for Store"** 버튼을 누릅니다.
4. **Android** 섹션에서 `Generate`를 누릅니다.
5. 옵션은 기본값 그대로 두고 `Download`를 누르세요.

## 3. 설치

- 다운로드된 파일 압축을 풀면 `.apk` 파일(signed-apk)이 있습니다.
- 이 파일을 핸드폰으로 옮겨서 설치하면 끝!

---
**💡 팁:** 만약 웹 호스팅이 어렵다면, 그냥 **핸드폰 Chrome 브라우저**에서 이 사이트에 접속한 뒤 **"홈 화면에 추가"**만 해도 APK 설치와 99% 똑같은 효과(아이콘 생성, 전체화면 실행)를 볼 수 있습니다!
