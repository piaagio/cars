# 특수차량 관리 시스템 사용 가이드

이 앱은 **서부산운용팀 전용**으로 제작된, **별도의 서버 없이** 작동하는 웹 앱입니다.

## 1. 주요 기능

### 🚗 차량 현황 확인 (대시보드)

- **전체/이동발전차/이동기지국** 탭을 눌러 차량 종류별로 모아볼 수 있습니다.
- **상태 색상**:
  - 🟢 **대여가능**: 현재 차고지에 있는 상태
  - 🔴 **사용중**: 누군가 대여해 간 상태
  - 🟡 **반납완료**: 반납 보고서 작성 후, 5분 대기 상태 (곧 '대여가능'으로 바뀜)

### 📝 대여 (출고) 방법

1. **[대여가능]** 차량을 클릭합니다.
2. **점검 등록** 화면이 나옵니다.
   - **팀명/운행자** 입력 (필수)
   - **출발 km** 입력
   - 각 항목(안테나, 공구, 청소 등) 점검 (양호/불량 체크)
   - 불량 발견 시 **사진 촬영** 버튼으로 증빙 사진 첨부
3. **[점검 완료 및 전송]** 버튼을 누릅니다.
4. **결과**:
   - 엑셀 파일(`.xlsx`)이 자동 생성되어 다운로드/공유됩니다.
   - 차량 상태가 **[사용중]**으로 바뀝니다.

### ↩️ 반납 방법

1. **[사용중]** 차량을 다시 클릭합니다. (또는 '반납완료' 버튼)
2. **반납 점검** 화면이 나옵니다.
   - **도착 km** 입력
   - **기름 잔량** 체크 (4/1 ~ 4/4)
   - 파손/특이사항 재점검
3. **[점검 완료 및 전송]** 버튼을 누릅니다.
4. **결과**:
   - 반납 보고서(엑셀)가 생성됩니다.
   - 차량 상태가 **[반납완료]**로 바뀝니다.
   - **5분 후** 자동으로 **[대여가능]** 상태로 초기화됩니다.

## 2. 자주 묻는 질문

**Q. 데이터는 어디에 저장되나요?**

- 이 앱은 **인터넷 서버**가 아닌, **사용자의 브라우저(핸드폰/PC)**에 저장됩니다.
- 따라서 앱을 삭제하거나 캐시를 지우면 데이터(차량 상태)가 초기화될 수 있습니다.

**Q. 엑셀 파일은 어디로 가나요?**

- **모바일**: 공유하기 창이 뜨며 카카오톡, 이메일 등으로 바로 보낼 수 있습니다.
- **PC**: '다운로드' 폴더에 저장됩니다.

**Q. 앱이 멈추는 것 같아요.**

- '반납' 처리 시 확인 창이 뜨거나, 이미지를 처리할 때 잠깐 멈출 수는 있으나 정상입니다.
- 계속 멈춘다면 앱을 새로고침 해주세요.
